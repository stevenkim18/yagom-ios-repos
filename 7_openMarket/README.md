## 프로젝트 설명
- 자유롭게 상품에 대한 정보를 올리고 조회할 수 있는 앱입니다.
- 서버와 연동됩니다.

## 핵심 경험
- JSON 데이터를 맵핑할 모델 설계
- CodingKeys 프로토콜 활용
- Postman 사용
- Http에 대한 이해
- 유닛 테스트의 이해 및 적용
- 네트워크와 무관한 테스트에 대한 이해 및 적용
- Mock, Stub 등에 대한 이해
- CollectionView에 대한 이해
